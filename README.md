A collection of basic C programming programs for beginners. This repository contains simple examples and practice problems to help learn the fundamentals of the C programming language.

## 📚 Topics Covered

- Hello World
- Variables and Data Types
- Input and Output
- Operators
- Conditional Statements (`if`, `if-else`, `switch`)
- Loops (`for`, `while`, `do-while`)
- Functions
- Arrays
- Strings
- Pointers
- Structures
- File Handling
- Recursion
- Pattern Programs
- Number Programs
- Basic Algorithms

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repository-name.git
   ```

2. Compile a program:
   ```bash
   gcc filename.c -o program
   ```

3. Run the program:
   ```bash
   ./program
   ```
   **Windows:**
   ```bash
   program.exe
   ```
